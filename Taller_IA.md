# Búsquedas No Informadas en Inteligencia Artificial

Este proyecto implementa los algoritmos BFS (Breadth First Search)
y DFS (Depth First Search) usando Python en Google Colab.

## Descripción

Se creó un grafo con los nodos A, B, C, D, E, F y G.

Se implementaron dos algoritmos:

- BFS (búsqueda en anchura)
- DFS (búsqueda en profundidad)

## Visualización

El programa muestra visualmente cómo se recorren los nodos del grafo.

Los nodos visitados cambian de color para observar el recorrido.

## Herramientas usadas

Python  
Google Colab  
NetworkX  
Matplotlib
